6. Размер: `Малый (< 10 сотрудников)`

8. Принять условия
9. **Поток данных:** выбрать **Веб**

   - Название потока: `FlowAi Main`

## 🎯 БЛОК 2. Google Ads (через 2 дня после Блока 1)

### Шаг 2.1. Создание аккаунта

1. https://ads.google.com → **Начать сейчас**
2. Войти через Google-аккаунт (рекомендую тот же, что и Analytics)
3. **Пропустить Smart Campaign** → ссылка внизу: `Switch to Expert Mode`
4. **Пропустить кампанию** → создать **только аккаунт** (кнопка `Create an account without a campaign`)
5. **Биллинг:**
   - Страна: `Казахстан`
   - Валюта: `KZT`
   - Часовой пояс: `Asia/Almaty`
6. **Платёжный профиль:** Индивидуальный предприниматель или Физ. лицо
7. Добавить банковскую карту → минимум 10 000 ₸ на балансе

### Шаг 2.2. Связать с GA4

1. В Google Ads: **Tools & Settings (🔧) → Measurement → Linked Accounts**
2. **Google Analytics (GA4)** → **Link**
3. Выбрать ресурс `flowai.kz` → **Link**
4. Включить **Auto-tagging** (Tools → Conversions → Settings)

### Шаг 2.3. Создать Conversion Action

1. **Tools → Goals → Conversions → Create**
2. **Import from GA4** → выбрать событие `whatsapp_click`
3. Название: `WhatsApp Click`
4. **Category:** `Submit lead form` (Google рассматривает как лид)
5. **Value:** `Не использовать значение` (или 5000 ₸ за клик как оценочное)
6. **Count:** `One` (только 1 конверсия с одного пользователя за сеанс)
7. **Conversion window:** 30 дней
8. Сохранить

### Шаг 2.4. Скопировать Conversion ID

После создания: `Tools → Conversions → WhatsApp Click` → вкладка **Tag setup** → скопировать `AW-XXXXXXXXX/YYYYYYY`.

**→ Пришлите этот ID в чат** — я настрою автоматический вызов конверсии при клике.

### Шаг 2.5. Первая кампания Search

1. **Campaigns → + New campaign**
2. Goal: **Sales** (НЕ выбирать конкретную цель — выбрать `Create a campaign without a goal's guidance`, так гибче)
3. Campaign type: **Search**
4. Settings:
   - **Campaign name:** `FlowAi | Search | WhatsApp-bot`
   - **Networks:** только `Google Search` (выключить Display Network и Search Partners на старте)
   - **Location:** `Kazakhstan`
   - **Language:** `Russian`, `Kazakh`
   - **Budget:** `2000 ₸/day`
   - **Bidding:** `Maximize conversions` (первые 2 недели без Target CPA)
   - **Ad schedule:** `Mon–Sun 9:00–22:00`
   - **Device targeting:** Mobile +20%
5. **Ad group 1: whatsapp-bot** — скопируйте ключевые слова и текст объявления из `ADS_COPY.md` (файл в этой же папке)
6. Создать ещё 5 Ad Groups (telegram-bot, ai-chatbot, voice-ai, business-automation, branded) по той же схеме
7. **Запустить кампанию** → проверка модерации 1–24 часа

## 🎯 БЛОК 3. Яндекс.Директ (после того как Google Ads работает 2 дня)

### Шаг 3.1. Создать аккаунт

1. https://direct.yandex.ru → **Запустить рекламу**
2. Использовать Яндекс-аккаунт (тот же, где создавали Метрику)
3. Страна: `Казахстан`, Валюта: `KZT`
4. Пополнить баланс: минимум **7 000 ₸** (рекомендую сразу 20 000 ₸)

### Шаг 3.2. Привязать Метрику

1. В интерфейсе: **Библиотека → Связь с другими продуктами Яндекса**
2. Добавить номер счётчика Метрики (из Блока 1.2)
3. ✅ Разрешить доступ

### Шаг 3.3. Настроить цель в Метрике (ВАЖНО — до создания кампании)

1. Открыть Метрику → **Настройка → Цели → Добавить цель**
2. **Тип условия:** `JavaScript-событие`
3. **Идентификатор цели:** `whatsapp_click`
4. **Название:** `Клик на WhatsApp`
5. Сохранить

### Шаг 3.4. Первая кампания

1. **Новая кампания → Текстово-графические объявления**
2. **Название:** `FlowAi | Поиск | WhatsApp-бот`
3. **Места показа:** только `Поиск` (РСЯ подключим через 2 недели)
4. **Стратегия:** `Оптимизация конверсий`
   - Цель: `Клик на WhatsApp`
   - Ограничение: `Стоимость конверсии` → `1 500 ₸`
   - Дневной бюджет: `1 500 ₸`
5. **Регион:** `Казахстан`
6. **Расписание:** `Пн-Вс 9:00-22:00`
7. **Минус-фразы** (общие на всю кампанию):
   ```
   бесплатный скачать отзывы что такое как работает wikipedia вики бесплатно
   своими руками самому python node js разработчик вакансия работа
   ```
8. Создать Ad Groups по образцу из `ADS_COPY.md` (Яндекс-секция)
